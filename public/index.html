Getting Started

DevUI is a modern CSS component library that provides ready-to-use components for building beautiful web interfaces quickly and efficiently.

Installation

Add DevUI to your project by including the CSS file in your HTML head:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My App</title> <link rel="stylesheet" href="devui-library.css"> </head> <body> <!-- Your content here --> </body> </html>

Preventing Style Conflicts

To prevent external CSS from affecting DevUI components, wrap your content in the devui-library class:

<body class="devui-library"> <button class="btn btn-primary">Protected Button</button> <span class="chip chip-success">Protected Chip</span> </body>

Customization

DevUI uses CSS custom properties (variables) for easy customization. Override these variables to match your brand:

:root { /* Colors */ --primary-color: #your-brand-color; --font-family-base: 'Your Font', sans-serif; --border-radius-md: 8px; /* Override any variable you need */ }
